# Ecommerce Website

A simple frontend ecommerce website built using HTML, CSS, and JavaScript.  
This project focuses on creating a clean UI for an online store and practicing basic web development concepts.

## Features
- Product listing interface
- Responsive layout
- Clean product display UI
- Basic frontend structure for an ecommerce platform

## Tech Stack
HTML  
CSS  
JavaScript  

## Project Structure
index.html – main webpage structure  
style.css – styling and layout  
script.js – basic frontend functionality  

## Status
Work in progress. Future improvements may include:

- Shopping cart functionality
- Product filtering
- Backend integration
- Payment system integration

## Purpose
This project was created as a practice project while learning frontend web development.
